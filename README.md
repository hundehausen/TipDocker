# TipDocker

Dockerize TipXMR

## Installation

### Dependencies

- git
- docker
- docker-compose
- wget

```bash
git clone https://github.com/hundehausen/TipDocker.git
cd scripts
./install_linux.sh
cd ..
docker-compose up
```

Then open [http://localhost:8080](http://localhost:8080) in your browser.
